# Gallery
Gallery: A chill wallpaper showcase slapped together with React, Tailwind, and Framer Motion. Pulls images straight from my wallpapers repo as a headless CMS. ¯\_(ツ)_/¯
Basically just a preview for github.com/revanthlol/wallpapers.

## Running Locally
```bash
# Clone it
git clone https://github.com/revanthlol/gallery.git
# Install deps
npm install
# Fire it up
npm run dev
```
